# Product Requirements Document

## Absolute Cinema

### 1. Product Overview

- **Product Name**: Absolute Cinema (AC)

- **Version**: 1.0.0

- **Product Type**: Front-end Interface

Absolute Cinema - This is a site for getting acquainted with the latest news in the film industry.

### 2. Target Users

- **Adults and Teenagers** - Movies and  Series
- **Kid** - Movies and Series only for kids

### 3. Core Features

#### 3.1 User Authentication and Autorization

- User Registartion - Account Creation (Opt. with Email verification.)
- User Login - Secure Authentication  with JWT tokens
- Password Management - Change password, forget/reset password functionality (Opt.)

### 4. Technical Specification

#### 4.1 Pages

- /login - Login Page
- /registartion - Registration Page
- /home - Page with all movies and series
- /movies - Page only for movies
- /movies/:id - Page only for one movie
- /series - Page only for series
- /series/:id - Page only for one serie
- /home/kids - Page with movies and series for kids
- /favorites - Page with user's favorite movies and series

#### 4.2 Layouts

- **Mobile** - starts with iPhone SE 
- **Desktop** - starts with FullHD (1920/1080)
- **Tablet** - starts with iPad Mini

### 5. Security Features
- **Authenticantion** - Email and Password verification

### 6. Succes Criteries

- FSD architechture
- Folder Structure
- Выгнать из группы Жавохира
- User-Friendly interface for all devices (Desktop and Mobiles)
- Show only latest updates in movies and series
- Secure preservation of favorite movies
